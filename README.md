# Lotto

Lottery number analyzer with frequency statistics, bar chart visualization, and random number drawing. Tracks historical draws and calculates occurrence probability for each number (1–49).

## Features

- Frequency analysis across all historical draws
- Color-coded bar chart visualization with mouse hover tooltips
- Random lottery number generator
- Binary file persistence for statistics data

Built with C++ and Borland VCL.
